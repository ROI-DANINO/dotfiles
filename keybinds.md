# Keybinds Cheat Sheet

> Mod = Super (Windows key)

---

## Niri ‚Äî App Launchers

| Bind | Action |
|------|--------|
| `Mod+T` | kitty (terminal) |
| `Mod+W` | default browser (zen) |
| `Mod+F` | thunar (GUI file manager) |
| `Mod+Slash` | walker (app launcher) |
| `Alt+M` | btm (system monitor) |

---

## Niri ‚Äî Session

| Bind | Action |
|------|--------|
| `Mod+Shift+L` | lock screen (hyprlock) |
| `Mod+Shift+E` | quit niri |
| `Mod+Shift+P` | power off monitors |
| `Mod+Shift+K` | toggle idle (off = no idle blank; on = start idle and blank now, then auto-lock at 10min) |
| `Mod+Shift+B` | toggle between waybar and sysbar |

---

## Niri ‚Äî Window Focus

| Bind | Action |
|------|--------|
| `Mod+H` / `Mod+Left` | focus column left |
| `Mod+L` / `Mod+Right` | focus column right |
| `Mod+J` / `Mod+Down` | focus window down |
| `Mod+K` / `Mod+Up` | focus window up |
| `Mod+Ctrl+H` / `Mod+Ctrl+Left` | focus first column |
| `Mod+Ctrl+L` / `Mod+Ctrl+Right` | focus last column |
| `Mod+Q` | close window |

---

## Niri ‚Äî Window Moving

| Bind | Action |
|------|--------|
| `Mod+Shift+H` / `Mod+Shift+Left` | move column left |
| `Mod+Shift+Right` | move column right |
| `Mod+Shift+J` / `Mod+Shift+Down` | move window down |
| `Mod+Shift+Up` | move window up |
| `Mod+Comma` | consume window into column |
| `Mod+Period` | expel window from column |

---

## Niri ‚Äî Window Sizing

| Bind | Action |
|------|--------|
| `Mod+R` | cycle column width (1/3 ‚Üí 1/2 ‚Üí 2/3) |
| `Mod+Shift+R` | cycle window height |
| `Mod+Ctrl+R` | reset window height |
| `Mod+M` | maximize column |
| `Mod+Shift+F` | fullscreen |
| `Mod+C` | center column |
| `Mod+Minus` | column width -10% |
| `Mod+Equal` | column width +10% |
| `Mod+Shift+Minus` | window height -10% |
| `Mod+Shift+Equal` | window height +10% |

---

## Niri ‚Äî Workspaces

| Bind | Action |
|------|--------|
| `Mod+U` / `Mod+Page_Up` | focus workspace up |
| `Mod+I` / `Mod+Page_Down` | focus workspace down |
| `Mod+Tab` | previous workspace |
| `Mod+1-9` | focus workspace N |
| `Mod+Ctrl+U` / `Mod+Ctrl+Page_Up` | move column to workspace up |
| `Mod+Ctrl+I` / `Mod+Ctrl+Page_Down` | move column to workspace down |
| `Mod+Shift+1-9` | move column to workspace N |
| `Mod+Shift+U` | move workspace up |
| `Mod+Shift+I` | move workspace down |

---

## Niri ‚Äî Screenshot

| Bind | Action |
|------|--------|
| `Ctrl+Space` / `Print` | interactive screenshot |
| `Ctrl+Print` | screenshot screen |
| `Alt+Print` | screenshot window |

---

## Niri ‚Äî Misc

| Bind | Action |
|------|--------|
| `Mod+Shift+Slash` | show hotkey overlay |
| `Mod+Space` | toggle keyboard language (us/Hebrew) |
| `Mod+G` | fix selected Hebrew/English wrong-layout text |
