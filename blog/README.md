## Style guardrails
- First-person, honest over polished; hero's-journey framing.
- Plain language; explain or avoid jargon.
- Every entry anchored to a real milestone or decision.
- Always end with a 🔗 LinkedIn cut — punchy, repost-ready, scroll-stopping first line.

## When an entry gets written
At the end of every phase (the "leave the docs true" exit ritual) and any time a
significant decision lands.
